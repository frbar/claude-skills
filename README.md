# Skills for Claude Code

This repository contains skills for Claude Code.

You can check out this repository as your `skills` repository in a Claude Code user profile on Windows at:

`C:\Users\<user>\.claude\skills\`

In other words, this repo is intended to live under the Claude profile directory:

`C:\Users\<user>\.claude\`

## Available skills

| Skill | Description |
|---|---|
| `adr` | Write Architecture Decision Records |
| `back-to-main` | Switch cleanly to main, handle pending changes, summarize what landed |
| `context-mapping` | Map bounded contexts with DDD patterns and Context Mapper DSL |
| `create-new-branch-and-prep-pr` | Create a new git branch and prepare a commit + PR message |
| `nestjs-red-team` | Adversarial security audit for NestJS / TypeORM / PostgreSQL codebases |
| `nestjs-pii-leak-audit` | Audit PII leaks in logs, Sentry, Datadog, traces — detects raw object logging, missing redaction, email/password in console |
| `notion-archive-todos` | Archive completed tasks from the Notion todos page |
| `notion-todo` | Add tasks to the personal Notion todos page |
| `vs-workspace-theme` | Configure VS Code workspace theme colors and title bar |
